PrimeNumberCheck - ASP .Net MVC application that includes web page and unit tests
AutoTestPrimeNumber - C# + Selenium webdriver automation tests

Created web page should be started from Index.cshtml page. This page is located in Views/Number/Index.cshtml.
Before run unit tests please make sure that you have installed NuGet package Microsoft.Asp.Net.WebPages.

In the AutoTestPrimeNumber might be needed to provide your localhost. It can be changed in the StartPage.cs file.
